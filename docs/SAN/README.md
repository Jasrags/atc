## Game Relevance

These charts are reference material for building realistic SAN approach/departure procedures:

- **STARs** define the arrival routes and fixes — use these to validate our nav fix positions and add missing waypoints
- **SIDs** define departure routes — future departure routing
- **Approach plates** show the final approach course, altitudes, and fixes — validates our runway 9/27 setup
- **Airport diagram** shows taxiway layout, gate positions, hold-short lines — validates our taxiway network
